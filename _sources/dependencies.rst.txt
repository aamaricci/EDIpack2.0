Dependencies
#########################

The `EDIpack2.0` library requires just two additional software to be
installed:

* SciFortran_: an open-source Fortran library to support mathematical
  and scientific software development. See related documentation. 

* MPI_: a distributed memory parallel communication layer with support
  to modern Fortran compiler. 

.. _SciFortran: https://github.com/SciFortran/SciFortran
.. _MPI: https://github.com/open-mpi/ompi
